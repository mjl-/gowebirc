Gowebirc usage

Introduction

Gowebirc is an irc client with a web frontend. The go program maintains irc connections to irc servers. It also serves html and irc updates to the web frontend. The web frontend is html, javascript/jquery and the updates are delivered through server-sent events (which are supported by most modern browsers, and do not require special connection tricks, they are just long-lived get requests). See the gowebirc program for usage information. See below for commands you can execute from the web frontend.

Shortcuts

The following shortcuts are bound on the whole document and can be used anywhere (with focus on any element):

ctrl + j
Focus the input line at the bottom.

The following shortcuts are bound to the input line only:

ctrl + n
Make the next session active. Same effect as clicking on the button right of the active one, or the first if there are no more.
ctrl + p
Make the previous session active.
ctrl + k
Switch to previous active session.
tab
Tab completion of nick in session. The first match is expanded.

Commands

The following commands can be run from the web frontend, from the input field at the bottom of the screen.

/connect [addr [nick]]
Start new irc connection to given addr (of form "host[:port]", default port is 6667). Your default nick can be overridden with the second parameter.
/reconnect
Drop the connection for the current window and redial it.
/join channel
Join the "channel", for example "/join #go-nuts".
/query nick
Open new window with "nick".
/disconnect
Disconnect from the connection for the current window.
/away message...
Set a new away message. If no parameter is given, you are no longer away.
/nick newnick
Request for a nick change to "newnick".
/umode mode
Change user mode by "mode", e.g. /umode +e.
/whois nick
Request information about "nick" from server using whois request.
/n or /names
Ask list of nicks in current channel.
/me message...
Send an "action" message that is typically shown as "* message..."
/notice message...
Send a message to current window with the irc "notice" command (by default, messages are sent with the "privmsg" command).
/msg target message...
Send message to "target" (a nick or channel). this is the same as typing a normal message in the window of "target".
/invite nick channel
Invite "nick" to join "channel".
/version or /time or /ping
Send a version/time/ping ctcp request to the current window.
/kick nick [message...]
Kick "nick" from the channel of the current window, with the optional "message" as reason.
/op nick, /voice nick, /ban nick, /deop nick, /devoice nick, /unban nick
Give op, voice, ban status to "nick", or the opposite.
/mode mode...
Set "mode" for the current window, e.g. /mode +snt.
/part
Leave the current channel.
/close
Close the current window.
/closeall
Close all windows for the current irc connection.
/topic text...
Set a new topic for the current channel.
